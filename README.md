# Sync Connect - Plataforma Elite de Nicaragua

Esta es la infraestructura tecnológica propietaria de **Sync Connect**, una plataforma optimizada para el mercado de Nicaragua con un diseño premium y funcionalidades de alto rendimiento de grado corporativo.

## 🛠️ Ecosistema Sync Connect
- **Sync Core Engine**: Motor de alto rendimiento para gestión de rutas, renderizado ultra-rápido y optimización SEO dinámica.
- **Firebase Infrastructure (Auth, Firestore, Storage)**: Gestión de identidades, base de datos en tiempo real y almacenamiento en la nube de nivel militar.
- **Sync Lab AI (Genkit)**: Inteligencia Artificial avanzada para copywriting, asistencia de ventas y construcción de sitios.
- **Elite UI Framework (ShadCN & Tailwind)**: Interfaz moderna con soporte total para Modo Claro/Oscuro y diseño adaptativo.
- **Automated Mailer (Nodemailer)**: Sistema de comunicaciones transaccionales premium.

## 🚀 Guía de Operaciones Críticas

### 1. Gestión de Identidades
Si experimentas dificultades en el acceso:
1. Accede a tu consola de administración de identidades.
2. Asegura que los dominios de confianza estén correctamente registrados.
3. Verifica que los métodos de Email, Google y Teléfono estén operativos.

### 2. Variables de Infraestructura
El sistema requiere las siguientes claves para operar el núcleo de IA y la base de datos:
- `FIREBASE_PROJECT_ID`: Identificador de proyecto.
- `FIREBASE_PRIVATE_KEY`: Llave de seguridad privada.
- `GEMINI_API_KEY`: Núcleo de inteligencia artificial.
