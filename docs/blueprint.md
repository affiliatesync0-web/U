# **App Name**: NicaAffiliate Connect

## Core Features:

- Affiliate Onboarding: Secure registration for affiliates, collecting personal, bank account details (number, Nicaraguan bank type, account holder, and Gmail).
- Product Catalog Viewer: Affiliates can browse available products/services, view their codes, prices, and commission rates.
- Sales Transaction Recorder: Affiliates can register new sales by entering product code, buyer's name, last name, and email. This automatically updates their balance.
- Admin Product Manager: Administrator interface to add, edit (code, name, price, commission), and delete products/services. Includes setting the specific Nicaraguan bank account for product payouts.
- Admin Overview Dashboard: A centralized dashboard for administrators to monitor all sales, view registered affiliates' information, and track overall performance.
- Affiliate Commission Tracking: Automatic calculation and display of the affiliate's current commission balance based on successfully recorded sales.
- Product Description Assistant: An AI-powered tool for administrators to generate engaging and optimized product descriptions based on initial inputs (product name, category, and features).

## Style Guidelines:

- Color scheme: Light mode for a professional and accessible feel. Primary brand color is a trustworthy deep blue (#2870A3), representing stability and growth. Background uses a very light blue-gray (#EFF2F4) to ensure readability and spaciousness. Accent elements and calls to action feature a vibrant violet (#A37EDC) for visual interest and highlighting key interactions.
- Headlines use 'Space Grotesk' (sans-serif) for a modern and tech-informed appearance. Body text uses 'Inter' (sans-serif) for high readability across various data displays and user inputs.
- Utilize a consistent set of clean, line-based icons for navigation and functional elements (e.g., user profiles, money transactions, product categories), maintaining a professional and intuitive interface.
- Implement a clear, modular layout with well-defined sections for dashboards and forms, optimizing for data display and ease of input. Ensure responsive design for seamless use on desktop and mobile devices, crucial for affiliates on the go.
- Incorporate subtle animations for state changes and user feedback, such as soft transitions on hover states for buttons, confirmation alerts on form submissions, and data loading indicators.
